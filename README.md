# Animaru Anitube UA Extension

Цей репозиторій містить розширення для **Animaru/Aniyomi**, яке дозволяє переглядати аніме з [anitube.ua](https://anitube.ua).

## Використання

1. Завантажити цей репозиторій.
2. Додати його в Animaru через "Extensions → Add Repository".
3. Після оновлення з’явиться джерело **Anitube UA**.
